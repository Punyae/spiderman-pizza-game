# Spider-Man Pizza Survival Game

A fast-paced, 2D arcade survival game built with HTML5 Canvas and JavaScript. Dodge the incoming Green Goblin pumpkin bombs, outrun him, and collect pizza slices to keep your timer alive!

## How to Play
* **Move:** Use `W-A-S-D` or the `Arrow Keys`.
* **Restart:** Press `R` on the Game Over screen to instantly try again.
